# auth-app-backend

A backend for [auth-app](https://github.com/thepicture/auth-app)

## Installation
`git clone https://github.com/thepicture/auth-app-backend.git`

`cd auth-app-backend`

`node index.js`

## Run

`google-chrome http://localhost:3002`
